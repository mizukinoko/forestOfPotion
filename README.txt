---------------------------------起動方法---------------------------------
ポーションの森.exeを実行してください

---------------------------------操作方法---------------------------------
移動       ：移動は「WASD」で操作出来ます。
攻撃       :「Enter」
走る       :「Space」を押しながらWASD
所持品の確認：「Eキー」で確認できます。
所持品の操作：「矢印キー」でアイテムを選択し「Enter」を押すと「使う」「捨てる」の選択肢を選べます※1。
　　　　　　　選択肢を閉じるときは「Eキー」を押します。
　　　　　　　アイテムウィンドウを閉じるときは「Eキー」を押します。
アイテム取得：アイテムが落ちている場所で一度立ち止まります。
　　　　　　　アイテムが落ちているところを通るだけではアイテムを取ることは出来ません。
